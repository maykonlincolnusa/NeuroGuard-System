# Roadmap do NeuroGuard

## Versão 0.1.0 (Atual)

### Funcionalidades Implementadas

- ✅ Leitura de arquivos EDF
- ✅ Pré-processamento de sinais EEG
- ✅ Extração de características
- ✅ Análise exploratória
- ✅ Modelos clássicos de ML
- ✅ Modelos de deep learning (CNN, LSTM, Transformer)
- ✅ Avaliação de modelos
- ✅ Anonimização de dados
- ✅ Criptografia de dados
- ✅ API REST básica
- ✅ Documentação inicial

## Versão 0.2.0 (Próxima)

### Planejado

- [ ] Autenticação e autorização na API
- [ ] Dashboard web para visualização
- [ ] Integração com bancos de dados (PostgreSQL)
- [ ] Cache Redis para performance
- [ ] Processamento assíncrono (Celery)
- [ ] Suporte a múltiplos formatos (FIF, BDF)
- [ ] Pipeline de ML automatizado (MLflow)
- [ ] Monitoramento e alertas (Prometheus/Grafana)
- [ ] Testes automatizados completos
- [ ] CI/CD pipeline
